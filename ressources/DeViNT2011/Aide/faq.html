Foire aux questions

    • Comment utilise t’on le clavier dans les projets Devint ?
    • En règle générale, la fonction des touches du clavier doit être :
      • flèches directionnelles pour se déplacer dans les menus,
      • ENTREE pour valider un choix,
      • ECHAP pour quitter un niveau ou terminer le jeu
      • F1, donne le but du jeu, F2, F3 … des informations diverses
      • CTRL, ESPACE, paramétrage, action.
    • Tous les projets respectent ils les règles d’utilisation du clavier ?
    • Non, certains parmi les plus anciens ont été conservés pour leur originalité bien qu’ils ne respectent pas ces contraintes d’interface.
    • Que faire si l’installation ne débute pas automatiquement?
    • Exécuter directement le fichier Install.exe sur le CD.
    • Que réalise la désinstallation ?
    • La désinstallation supprime d’un coup tous les projets installés, ainsi que les icônes ‘Jeu Devint’ et ‘Aide Devint’.
    • Comment installer les périphériques de certains jeux?
    • Le dossier ‘install’ du CD contient plusieurs logiciels d’installation, pour le tapis de danse, pour la Wiimote, … qu’il faut exécuter si on le souhaite, sans garantie de résultat.
    • Comment désinstaller les périphériques supplémentaires installés?
    • Il faut aller dans le menu ‘Ajout/Suppression de Programmes’ et cliquer sur ‘Modifier/Supprimer’.
    • Comment désinstaller les projets Devint ?
    • Pour désinstaller les projets Devint, on peut faire comme ci-dessus, également exécuter le fichier Uninstall.exe dans le répertoire des projets.
    • Comment afficher et exécuter les projets installés?
    • En cliquant sur l’icône 'Jeu Devint’ du bureau, la liste des projets installés apparaît, la synthèse vocale lit leurs noms, et la touche Enter ou un clic de la souris permet de les exécuter.
    • Que contient la fiche d’aide de chaque projet Devint ?
    • La fiche du projet Devint indique
      • le titre, l’année, et les auteurs du projet
      • le public concerné, avec les mentions MV pour mal voyant, et NV pour non voyant, ainsi que la classe d’âge concernée
      • les règles du jeu, et la façon de jouer (gameplay)
      • les notes d’utilisation, qui précisent les matériels et logiciels requis pour utiliser le jeu, et tous les problèmes d’utilisation déjà répertoriés.
    • Faut-il redémarrer l’ordinateur pour installer les projets Devint ?
    • Non, l’installation est une copie du CD dans le dossier indiqué.
    • Que faire si la synthèse fonctionne et lit les noms des jeux, mais que certains jeux ne fonctionnent pas, sous Windows Seven ?
    • Réinstaller le CD dans un dossier où vous avez les droits d’écriture, comme par exemple le dossier Mes Documents ou Documents.
    • Que faut il installer pour utiliser le tapis de danse Stepmania ou la guitare de Guitar Hero ?
    • Il faut exécuter le fichier EMS.exe qui se trouve dans le dossier install du CD.
    • Comment utiliser la WiiMote pour le jeu CallOfPolytech ?
    • Il faut exécuter le fichier SetWiiRemoteJ.exe du dossier install et utiliser une clé BlueTooth, mais on ne garantit pas le résultat.
    • Que faire sans tapis, Wiimote, ou périphérique de jeu ?
    • Tous les jeux fonctionnent avec le clavier ou la souris, consulter l’aide en ligne pour savoir comment.
    • Comment consulter l’aide en ligne ?
    • Il suffit de cliquer sur l’icône AideDevint sur le bureau et l’Aide de tous les projets installés est affichée à la demande dans un navigateur web.
    • Quelle poignée de jeu utiliser pour le jeu Invade2.20 ?
    • La poignée SAITEK ST 90 (à défaut, on utilise le clavier).
    • Peut-on utiliser le tapis USB dans les jeux Devint?
    • Oui, dans le jeu TapTitTaupe par exemple.
    • Faut-il installer tous les jeux du CD à la fois ?
    • C’est possible, mais on conseille d’en installer plutôt cinq à dix, car au delà il est difficile de gérer le menu de choix des jeux qui en affiche cinq.
    • Que faire si le menu de lancement des projets ne réagit pas au clavier ?
    • Cela arrive surtout lors de la première installation, il suffit de cliquer sur un projet, et d’en sortir avec la touche Echap.
